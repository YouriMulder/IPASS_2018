Increase the currentCentury using centuryIncreaseValue.

currentCentury += centuryIncreaseValue

Warning

Check if the given century is a valid value.

Parameters
centurythe value you want to check if it's valid
Returns
A bool containing 1 when the given year valid and 0 when it's not valid.

This methods check whether the given century is valid. This is basicly a check if the last two digits are zeroes.
For example:
2200 = valid
2202 = invalid
101200 = valid
101210 = invalid

◆ currentCentury

int time::currentCentury = 2000
static

This is the current century of the real-time clock you might use.

This is a shared variable between instances. To return a correct year value the real-time clock has to know the current century.

Warning
Changing this means every connected real-time clock his century will increase.
